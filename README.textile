h1. Nuvola Academy

Nuvola Academy is an online e-Learning portal built on Ruby on Rails.  It allows for the upload and play back of videos via JW Player. You can create courses designed for resale.  Current payment gateway we are using is Stripe.


h2. Ruby on Rails

This application requires:

* Ruby version 1.9.3
* Rails version 3.2.13

Learn more about "Installing Rails":http://railsapps.github.io/installing-rails.html.

h2. Database

This application uses SQLite with ActiveRecord.

h2. Development

* Template Engine: ERB
* Testing Framework: RSpec and Factory Girl and Cucumber
* Front-end Framework: Twitter Bootstrap (Sass)
* Form Builder: SimpleForm
* Authentication: Devise
* Authorization: CanCan
